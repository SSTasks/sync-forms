Ultimate app for interviewing candidates for a QC position using interactive forms.

To begin an interview, wait until an expert runs the interview session.

How to use the app:

  • Choose an accessible interview from the list of interviews.
  • Interact with form controls to identify problems. Have fun!

The application main features are:

  • Create custom forms for interview and save them.
  • Ability to create an interview with chosen form.
  • Ability to mimic majority of events from one user to another.
  • Possibility of creation, editing, deleting user and groups. Assigning users to groups and choose user's permissions.
  • Possibility of viewing interview statistics: login/logout time, interview's actions and their performer, interview start and end time.
